# DOCS

Go to /docs to see docs and endpoints.

# SETUP

Docker container:

- docker-compose up -d

Linux:

- sudo apt install python3-dev libpq-dev

Pip commands:

- pip install -r requirements.txt

# Execute to run:

- python3 main.py

or

- uvicorn main:app --reload

or

- python3 -m uvicorn main:app --reload
